# 이진 탐색 트리

트리라고 다 되는거 아님 이진 탐색 트리의 경우 가능

* 부모 노드 보다 왼쪽 자식 노드가 작다
* 부모 노드 보다 오른쪽 자식 노드가 크다

왼쪽 자식 노드 < 부모 노드 < 오른쪽 자식 노드

이건 알고리즘이라기 보다 자료 구조에 가깝고 출제 빈도가 낮아 구현 패스

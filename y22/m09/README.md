데이트도 많이 못 했는데 다른 것도 딱히 뭔가 한 것 같지 않다.

차라리 데이트만 했다면 덜 억울했을 것 같다. 

더 붙어 있고 싶은데 시간이랑 에너지가 부족하다.

체력을 더 키우던가 해야지.

오늘만 코테 두개를 봤다. 근데 자괴감이 들었다.

하나는 11번가 코테 였고 하나는 프로그래머스 코테였다.

11번가 코테는 진짜 이거로 분별이 될까 싶은 프로그래머스 1단계 정도의 문제들만 나왔다.

영어로 나와서 좀 어색해서 그렇지 문제 자체는 쉬웠다.

하나는 이해가 아예 안되서 검색해서 문제 이해하고 내 풀이로 풀었다.

SQL 문제는 아예 못 풀었다.

어려운게 아니라. SQL 로 이런 것도 되네? 느낌이었다.

프로그래머스 코테는 저번이랑 기출이랑 난이도는 비슷했던 것 같다.

SQL, 1단계, 2단계, 3단계 이런 느낌이 었다.

SQL, 1단계는 10~20분 정도 만에 푼 것 같고 

나머지 시간을 두 문제에 쏟았는데 전략도 실패하고, 내 실력도 부족했다.

2단계 건너뛰고 3단계를 풀었다.

그러다가 30분 정도 보내고, 못 풀겠어서 다시 2단계로 넘어 왔다.

근데 그 2단계도 구현하다가 애먹어서 테스트도 안 돌려보고, 제출도 못했다.

실력도 부족했지만, 전략도 안 좋았다.

끝나고도 짜증나서 2번 문제 구현하던거 테스트 케이스 복사해 두고 끝까지 풀었다.

BFS 를 하는데 뇌정지가 와서 이상한 BFS 가 됐다.

시간이 오래 걸려서 그러지 문제는 다 풀만 하다. 

물론 3단계는 더 오래 걸릴 것 같다.

아무튼 제시간에 문제를 푸는 연습을 해야겠다.

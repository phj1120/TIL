## 여행 경로

[여행 경로](https://school.programmers.co.kr/learn/courses/30/lessons/43164#)

말로 풀어봐도 감이 잡힐듯 하면서도 모르겠다. 왜 저게 방문 경로지?

```
스택 = [시작점]
경로 = []
스택이 빌 때까지 반복
    다음 값 = 스택.peek()
    다음 값에 해당하는 티켓이 없을 경우
        경로 = 스택.pop()
    다음 값에 해당하는 티켓이 있을 경우
        스택.push(다음 여행 도착지)
경로.reverse()
```

코드는 알겠는데 왜 그게 경로가 되는지 로직을 이해 못 했다.
